# PakkiBird

A Flappy Bird-inspired browser game built with TypeScript, Canvas API, and Vite.
Procedurally generated assets - zero external images or sounds!

## Running Locally

1. `npm install`
2. `npm run dev`

## Deployment

Pushes to the `main` branch are automatically deployed to GitHub Pages via GitHub Actions.
Make sure to go to Settings -> Pages and set the source to `gh-pages` branch.

## Update Credits
Edit `src/config/CreditsConfig.ts` to add/remove team members.

## Add Changelog Entries
Edit `src/config/ChangelogConfig.ts` to add release versions.

## Cheat Codes
Enter these anywhere in the game:
- NOCLIP: Up Up Down Down Left Right Left Right B A
- CHAOS_MODE: C H A O S
- SLOW_MO: S S S
- BIG_BIRD: B B B
- SCORE_HACK: S C O R E
